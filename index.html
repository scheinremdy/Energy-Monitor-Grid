Project Overview

This project simulates grid load insights, showcasing both actual and forecasted energy consumption. You can select the country to see specific grid load trends based on that country's historical energy usage data.

Project Help

This dashboard provides real-time and forecasted grid load data. Here's what the charts represent:

  • Actual Grid Load: Represents current energy usage of the grid.
  • Forecasted Grid Load: Predicted future energy consumption based on trends.
  • Date Range Filter: Adjust the date range to explore grid load and forecasts.
  • Country Selector: Choose a country to see relevant grid load patterns.

Why is this important? Monitoring grid load helps ensure stable energy supply and efficient management.
